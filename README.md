# BYD Electric Vehicles Website

A modern, responsive website showcasing BYD's electric vehicle lineup and technology.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Interactive Elements**: Hover effects, scroll animations, and dynamic content
- **Vehicle Showcase**: Detailed information about BYD's electric vehicle models
- **Technology Section**: Highlights BYD's innovative EV technology
- **Contact Form**: Functional contact form with validation
- **SEO Optimized**: Semantic HTML5 structure with proper meta tags

## Vehicle Models Featured

1. **BYD Han** - Premium electric sedan
2. **BYD Tang** - 7-seater electric SUV
3. **BYD Seal** - Sporty performance sedan
4. **BYD Dolphin** - Compact electric hatchback
5. **BYD Atto 3** - Compact electric SUV
6. **BYD Song Plus** - Mid-size electric SUV

## Technology Highlights

- **Blade Battery** - Revolutionary lithium iron phosphate battery technology
- **e-Platform 3.0** - Integrated electric vehicle platform
- **Zero Emissions** - True zero-emission electric vehicles
- **Fast Charging** - Up to 150kW DC fast charging support

## Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No additional setup required - it's a static website
